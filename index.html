About the DataTracker

Every month I run into the same problem - I'm almost out of data on my cellphone. The DataTracker will keep track of how much data I use within the month on my cellphone, and hopefully, preventing me from going over on my data. It will be able to see how much data I've used so far, how much I have left in the month, along with what applications are using the most data.

What the App Will Do

  • Allow you to input how much data you have used during the day
  • Allow you to input how much each application will use
  • Allow you to view how much data you have used and how much you have left
  • Tell you how to budget your data for the rest of the month
